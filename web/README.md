# Silicon Valley Locksmith Website

A professional, modern one-page website for Silicon Valley Locksmith showcasing automotive, residential, and commercial locksmith services.

## Features

- **Sticky Call Button**: Always-visible call-to-action for immediate contact
- **Hero Section**: Eye-catching introduction with service overview and special offer
- **Service Sections**: Detailed information about residential, commercial, and automotive services
- **Service Areas**: Comprehensive list of all covered Silicon Valley locations
- **Contact Information**: Multiple ways to reach the business (phone, email, hours)
- **Responsive Design**: Optimized for mobile, tablet, and desktop viewing
- **SEO Optimized**: Proper meta tags and semantic HTML structure

## Logo Requirements

For optimal display, ensure your logo meets these specifications:

- **Dimensions**: 512x512 pixels (square)
- **Format**: PNG with transparency
- **File Size**: Maximum 500 KB (recommended 50-200 KB)
- **Location**: Place in `web/public/logo.png`

### Error Handling & Fallback

The website includes automatic logo fallback:
- **Primary**: Loads `logo.png` (optimized raster image)
- **Fallback**: Uses `logo.svg` if PNG fails to load
- **Console Warnings**: Alerts if image loading fails

For detailed optimization instructions, see: `web/public/IMAGE_OPTIMIZATION_GUIDE.md`

## Technology Stack

- **Framework**: React 18 with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Icons**: Lucide React
- **Package Manager**: Yarn
